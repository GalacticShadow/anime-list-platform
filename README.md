# Anime List Platform

## Overview

Anime List Platform is a full-stack web application for tracking anime and manga, viewing user profiles, managing ratings, and building personalized list experiences.

The project is designed as a modern anime community platform inspired by list tracking, profile customization, discovery, and social discussion features.

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Supabase
- PostgreSQL
- Git/GitHub

## Features

- Public username-based profile pages
- Profile header and profile stats
- Reusable component structure
- Dashboard and homepage sections
- Authentication-related flow planning
- Planned anime/manga list system
- Planned rating and Top 10 showcase features
- Planned community debate and discovery features

## What I Worked On

- Connected frontend components to Supabase-backed data structures
